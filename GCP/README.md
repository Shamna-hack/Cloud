# Infrastructure as Code (IaC) with Terraform
# Objective
Using Google Cloud the following tasks will be performed:
- Verify Terraform installation
- Define Google Cloud as the provider
- Create, change and destroy Google Cloud resources by using Terraform

## Step 1
On the Google Cloud Menu, click **Activate Cloud Shell** and the cloud shell will be opened.  
Confirm that Terraform is installed by running the command: ```terraform --version```
